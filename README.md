# Marketing Campaign Effectiveness Dashboard

## 📌 Project Overview
This project focuses on analyzing and optimizing marketing campaign performance using data-driven insights.  
The goal is to clean, transform, and visualize marketing campaign data to evaluate campaign effectiveness and support better business decisions.

The project workflow includes:
- **Data Cleaning** in Excel
- **Data Transformation & Querying** in SQL
- **Interactive Dashboard Creation** in Power BI

---

## 🎯 Objectives
- Measure the performance of marketing campaigns
- Identify high-performing and low-performing campaigns
- Analyze customer response patterns
- Track conversion metrics and ROI
- Provide actionable insights for marketing optimization

---

## 🛠️ Tools & Technologies Used
- **Microsoft Excel** – Data cleaning and preprocessing
- **SQL** – Data transformation and analysis
- **Power BI** – Dashboard creation and visualization
- **CSV Dataset** – Source data (`Marketing campaign dataset.csv`)

---

## 🔄 Project Workflow

### 1. Data Cleaning in Excel
- Removed duplicate records
- Handled missing/null values
- Standardized column names
- Corrected inconsistent data formats
- Checked data quality before analysis

### 2. Data Transformation in SQL
- Imported cleaned dataset into SQL
- Wrote queries for:
  - Customer segmentation
  - Campaign response analysis
  - Purchase behavior analysis
  - Conversion rate calculations
  - Revenue contribution insights

### 3. Dashboard Development in Power BI
Created an interactive dashboard with:
- Campaign performance KPIs
- Customer demographics analysis
- Conversion rate visuals
- Sales and revenue trends
- Response rate comparison by campaign
- Filters and slicers for dynamic exploration

---

## 📊 Key Metrics Analyzed
- Total Customers
- Total Campaigns
- Conversion Rate
- Response Rate
- Revenue Generated
- Average Customer Spend
- Purchase Frequency
- Campaign-wise Performance

---

## 📈 Dashboard Features
- Interactive filters and slicers
- KPI cards for quick insights
- Bar charts and pie charts for campaign comparison
- Trend analysis for customer purchases
- Demographic breakdown of customer behavior

---

## 🔍 Insights & Findings
Some example insights from the analysis may include:
- Certain campaigns performed significantly better than others
- Customers with higher income showed stronger purchase behavior
- Web visits influenced campaign response rates
- Specific customer segments had higher conversion potential
- Marketing spend can be optimized by focusing on high-performing campaigns
